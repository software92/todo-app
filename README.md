# To Do App v2 (React)

## stack

- react hook
- styled-components
- react-beautiful-dnd
- gh-pages

## 기능

- 할 일 추가
- 할 일 -> 할 일 완료 목록 이동
- 할 일 제거
- 할 일 <-> 할 일 완료 목록 drag 이동
